# sales_dashboard
Sales Dashboard 📊

An interactive Sales Dashboard built in Microsoft Excel to analyze and monitor business sales performance using dynamic charts, KPIs, and slicers. This dashboard provides insights into sales trends, representative performance, product contribution, and city-wise sales distribution.

🚀 Features
Dynamic and interactive dashboard
KPI Cards for:
Total Sales
Total Orders
Average Order Value
Total Discount
Monthly Sales Analysis
Sales by Sales Representative
Sales by Category
Top Products Analysis
Sales by City
Interactive Slicers for:
Product
Sales Representative
City
Month
🛠 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning & Formatting
📈 Dashboard Insights

The dashboard helps in:

Tracking overall sales performance
Identifying top-performing products
Comparing sales representatives
Analyzing city-wise sales trends
Monitoring monthly growth patterns
Evaluating discounts and average order values
📂 Dataset Information

The dataset contains sales transaction data including:

Product Name
Sales Representative
City
Order Details
Sales Amount
Discount
Order Quantity
Monthly Sales Data
🎯 Project Objective

The objective of this project is to create a visually appealing and user-friendly sales dashboard that enables quick business decision-making through interactive data visualization.
